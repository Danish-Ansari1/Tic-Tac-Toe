A sleek, modern, and full-stack implementation of the classic Tic-Tac-Toe game. This project features a **Python (Flask)** backend and a **Vanilla JavaScript** frontend with premium glassmorphism aesthetics.

---

## ✨ Features
- **Modern UI:** Clean, dark-themed design with smooth CSS animations.
- **Full-Stack Logic:** Backend handles server-side processes while the frontend provides an interactive UI.
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile screens.
- **Instant Feedback:** Real-time turn indicators and win/draw modal notifications.
